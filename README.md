Frontend Task (Next.js + React)

This project is designed to implement a frontend task using Next.js / React.

According to the requirements, it includes the following sections:

Dashboard based on DummyJSON with Chakra UI
(Login / Users / Products)

Games List Page with filters and a Single Game Details Page
powered by the RAWG API

Advanced Dropdown Select Component
built with Headless UI (Listbox) and Tailwind CSS

Tech Stack

Next.js (App Router)

React

JavaScript + JSX (no TypeScript)

Chakra UI (for the dashboard)

Tailwind CSS (for the advanced Select component)

RAWG API (Games)

DummyJSON API (Authentication / Users / Products)

Prerequisites

Node.js version 18 or higher

npm or yarn


Setup & Run
npm install
